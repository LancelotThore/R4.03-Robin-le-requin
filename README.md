# 🦷 M'T DENTS – Robin le requin

⭐ Projet universitaire réalisé en **2ᵉ année de BUT MMI**

Application web éducative destinée aux **enfants de 3 à 6 ans** pour apprendre les bonnes pratiques de **brossage des dents** de manière ludique et interactive.

L'application propose un **chronomètre de brossage**, des **fiches d'information pédagogiques** et un **quiz interactif**.



# 📱 Aperçu

Application web conçue **exclusivement pour une utilisation sur tablette et mobile**, conformément aux consignes du projet et aux maquettes Figma fournies.

# 🎯 Objectif du projet

Ce projet avait pour objectif de concevoir une **application pédagogique interactive** permettant aux jeunes enfants de :

* apprendre l'importance du brossage des dents
* comprendre certains concepts de santé bucco-dentaire
* adopter de bonnes habitudes dès le plus jeune âge

L’application a été développée à partir de **maquettes Figma réalisées par des étudiants de 3ᵉ année**.



# ✨ Fonctionnalités

* 👤 Création de **profils enfants**
* ⏱️ **Chronomètre de 2 minutes** pour accompagner le brossage des dents
* 📚 **Fiches d'information** pédagogiques :

  * les caries
  * les dents de lait
  * l'hygiène dentaire
* ❓ **Quiz interactif (QCM)** pour tester les connaissances
* 📱 Interface adaptée **tablette et mobile**



# 🚧 Fonctionnalités non implémentées

Lors de la phase de conception, un **suivi du brossage via un calendrier** avait été envisagé afin de permettre aux enfants de visualiser leurs habitudes de brossage au fil des jours.

Cependant, cette fonctionnalité n’a pas été implémentée dans la version finale du projet pour deux raisons :

* les **maquettes correspondantes n’étaient pas disponibles sur Figma**
* les **contraintes de temps du projet** ne permettaient pas de concevoir et développer cette fonctionnalité en plus du reste de l’application



# 🛠️ Technologies utilisées

* **React**
* **Vite**
* **TailwindCSS**

L’application est **100% front-end**.

Les données (profils, progression, etc.) sont **stockées uniquement dans le cache du navigateur** et ne sont pas persistées sur un serveur.
